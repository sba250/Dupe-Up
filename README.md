# Dupify

## We created this project with the purpose of making the search for make up dupes faster and easier

## Inspiration
Wanting to find cheaper yet quality make up products without having to surf the whole internet

## What it does
When users search for their favourit product, our website returns the top 6 closest matched products.

## How we built it
We built it using html, css, and javascript. 

## How to run
Clone the repo. Open the files in vscode. Then, click the live button on the bottom left bar. You should be able to use the website right now in your browser.

## Challenges we ran into
Learning how to implement an API as well as how to integrate code while we are both pushing to the website

## Accomplishments that we're proud of
Completing this product in a short period of time.

## What we learned
How to use API's, web development, and debugging.

## What's next for Dupify
Improving the interface design, including pictures for the recommendations, etc.
